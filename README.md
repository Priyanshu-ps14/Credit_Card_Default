# 💳 Credit Card Default Prediction using Machine Learning

## 📌 Project Overview

This project predicts whether a customer is likely to default on a credit card payment using Machine Learning. It uses customer financial information such as income, age, loan amount, and loan-to-income ratio to classify customers into default and non-default categories.

This project was completed as part of the **YBI Foundation Artificial Intelligence & Generative AI Internship Program**.

---

## 🎯 Objective

The main objectives of this project are:

- Understand the credit card default dataset.
- Perform data exploration and preprocessing.
- Train a Machine Learning classification model.
- Predict whether a customer will default.
- Evaluate the model using standard performance metrics.

---

## 📂 Dataset

The dataset contains **2000 customer records** with the following features:

- Income
- Age
- Loan
- Loan to Income Ratio
- Default (Target Variable)

Target Values:

- **0** → Customer will not default.
- **1** → Customer will default.
